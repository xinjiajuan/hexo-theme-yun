# Icons

The default icon used by the current theme (corresponding to the cdn.js.async in the file _config.yml (opens new window) .iconfont` link)

Most of the icons are taken from Remix Icon (opens new window) and keep their original names.

Only the most commonly used icons are provided by default, which is conducive to satisfying the use of the premise and reducing the loaded resources. If you want to use more icons, it is recommended to go to iconfont (opens new window) to customize your icon set. It is introduced in [head](/guide/config.html#head-head resources).

When writing your icon name in _config,yml, remember to prefix it with icon-. This distinguishes iconfont’s SVG icons from other font icons.

For Example：

icon: icon-github-line
